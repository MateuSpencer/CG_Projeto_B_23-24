# Computer Graphics — Crane (Project B)

**Course:** Computer Graphics (CG) — IST, University of Lisbon
**Year:** 2023/2024

## Description

An interactive **3D crane simulation** built with Three.js. Features a fully articulated crane with boom rotation, trolley movement, cable extension, and a 4-finger claw mechanism. Includes collision detection for picking up objects.

### Features

- Articulated crane: rotating boom, sliding trolley, extending cable, opening/closing claw
- Multiple camera views (including a hook-mounted camera)
- Wireframe toggle mode
- Keyboard-driven controls
- Collision detection between claw and load objects
- Smooth animation states

## Technologies

- **Language:** JavaScript
- **Library:** Three.js (WebGL)
- **Rendering:** HTML5 Canvas

## How to Run

Open `index.html` in a web browser, or serve with a local HTTP server:

```bash
python3 -m http.server 8000
# Open http://localhost:8000
```
